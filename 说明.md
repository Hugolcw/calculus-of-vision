## 六、常见问题

### Q1: 视频无法播放？

**A**: 确保使用支持MP4的播放器（如VLC、PotPlayer、Windows Media Player）

### Q2: 代码无法运行？

**A**:

1. 检查Python版本（需3.8+）：`python --version`
2. 检查是否安装了Manim：`pip install manim`
3. 检查是否安装了LaTeX（用于数学公式渲染）
4. 检查是否安装了numpy：`pip install numpy`

### Q3: 渲染速度慢？

**A**:

- Scene 4的3D场景渲染较慢是正常的（需要处理大量顶点）
- 可以使用 `-pql`先预览低质量版本（480P，快速）
- 最终渲染使用 `-pqh`获得高质量（1080P）

### Q4: LaTeX错误？

**A**:

- 确保安装了完整的LaTeX发行版
- Windows推荐MiKTeX（自动安装缺失的包）
- Linux/Mac推荐TeX Live
