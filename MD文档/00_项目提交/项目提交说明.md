## 九、常见问题

### Q1: 视频无法播放？
**A**: 确保使用支持 MP4 的播放器（如 VLC、PotPlayer、Windows Media Player）

### Q2: 代码无法运行？
**A**: 
1. 检查 Python 版本（需 3.8+）：`python --version`
2. 检查是否安装了 Manim：`pip install manim`
3. 检查是否安装了 LaTeX（用于数学公式渲染）
4. 检查是否安装了 numpy：`pip install numpy`

### Q3: 渲染速度慢？
**A**: 
- Scene 4 的 3D 场景渲染较慢是正常的（需要处理大量顶点）
- 可以使用 `-pql` 先预览低质量版本（480P，快速）
- 最终渲染使用 `-pqh` 获得高质量（1080P）
- Scene 4 已进行降采样优化（`[::10, ::10]`），防止渲染卡死

### Q4: LaTeX 错误？
**A**: 
- 确保安装了完整的 LaTeX 发行版
- Windows 推荐 MiKTeX（自动安装缺失的包）
- Linux/Mac 推荐 TeX Live
- 如果出现公式渲染错误，检查 LaTeX 环境变量配置

### Q5: Scene 4 渲染卡死？
**A**: 
- 代码中已包含降采样处理（`get_downsampled_array` 函数）
- 如果仍卡死，可以进一步增加降采样率（修改 `rate` 参数）
- 建议 Scene 4 的分辨率不超过 20×20

### Q6: 如何单独测试某个场景？
**A**: 
在 `sobel_v12_full.py` 中，可以单独运行某个场景类进行测试：
```python
# V12 使用独立的场景类，可以直接渲染单个场景
# 例如，只渲染 Scene 4：
manim -pql sobel_v12_full.py Scene4Vision

# 或者渲染完整视频（所有场景）：
manim -pqh sobel_v12_full.py FullSobelVideo
```
