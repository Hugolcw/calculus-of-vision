## 三、文件使用指南

### 3.1 观看视频

1. 直接双击 `.mp4` 文件即可播放
2. 推荐使用 VLC、PotPlayer 等播放器以获得最佳体验

### 3.2 查看文档

1. **Markdown 文件**：
   - 使用 VS Code、Typora、MarkdownPad 等工具打开
   - 或在 GitHub 上直接查看

2. **Python 代码**：
   - 使用 VS Code、PyCharm 等 IDE 打开
   - 需要 Python 3.10+ 和 Manim 环境

### 3.3 重新渲染视频

如果需要修改代码并重新渲染：

1. **安装环境**：
   ```bash
   pip install manim numpy
   ```

2. **安装 LaTeX**（用于数学公式渲染）：
   - Windows: MiKTeX
   - Linux/Mac: TeX Live

3. **渲染视频**：
   ```bash
   # 低质量预览（快速）
   manim -pql [文件名].py SobelUniverse
   
   # 高质量渲染（最终版本）
   manim -pqh [文件名].py SobelUniverse
   ```

## 六、常见问题

### Q1: 视频无法播放？
**A**: 确保使用支持 MP4 的播放器（如 VLC、PotPlayer）

### Q2: 代码无法运行？
**A**: 
1. 检查 Python 版本（需 3.8+）
2. 检查是否安装了 Manim：`pip install manim`
3. 检查是否安装了 LaTeX（用于数学公式）

### Q3: 渲染速度慢？
**A**: 
- Scene 4 的 3D 场景渲染较慢是正常的
- 可以使用 `-pql` 先预览低质量版本
- 最终渲染使用 `-pqh` 获得高质量

### Q4: LaTeX 错误？
**A**: 
- 确保安装了完整的 LaTeX 发行版
- Windows 推荐 MiKTeX
- Linux/Mac 推荐 TeX Live
